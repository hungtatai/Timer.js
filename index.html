Usage

Timer.once

Timer.once([name,] callback, interval)

Execute callback once after interval millisecond.

  • if name has not been given, name = (a unique id)
  • return a timer object

Timer.repeat

Timer.repeat([name,] callback, interval, [repeat_times]);

Execute callback repeat_times times every interval millisecond.

  • if name has not been given, name = (a unique id)
  • if repeat_times has not been given, repeat_time = Infinity
  • return a timer object

Timer.stop

Timer.stop(name)

Tools

Timer.stopAll

Timer.stopAll()

Stop all timer.

Timer.isRunning

Timer.isRunning(name)

Check timer's running status.

Sample

var timer1 = Timer.repeat('timer1', function(){
  console.log(this); // {name: 'timer1', repeat_times: ... ,stop: function(){ ...} } 
  console.log(new Date());
}, 1000);

console.log(timer1); // {name: 'timer1', repeat_times: ... ,stop: function(){ ...} }
